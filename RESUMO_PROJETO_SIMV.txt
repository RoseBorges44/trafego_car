TITULO DO PROJETO
-----------------
SIMV - Sistema Inteligente de Monitoramento Veicular


OBJETIVO
--------
Desenvolver uma solucao de visao computacional para monitoramento de trafego
urbano em tempo real, utilizando inteligencia artificial para deteccao,
rastreamento e contagem automatica de veiculos, com sistema de alertas
integrado para apoio a seguranca publica.


PROBLEMA IDENTIFICADO
---------------------
- Contagem manual de veiculos e ineficiente e imprecisa
- Falta de dados em tempo real sobre fluxo de trafego
- Dificuldade em identificar veiculos suspeitos rapidamente
- Ausencia de historico para planejamento urbano


SOLUCAO PROPOSTA
----------------
Sistema baseado em IA que processa video em tempo real para:
- Detectar e contar veiculos automaticamente
- Classificar veiculos por cor
- Identificar tipo de transito (livre, leve, moderado, intenso, congestionado)
- Emitir alertas para veiculos suspeitos via Telegram
- Gerar relatorios em PDF para tomada de decisao


TECNOLOGIAS UTILIZADAS
----------------------
| Funcao                | Tecnologia                    |
|-----------------------|-------------------------------|
| Deteccao de Objetos   | YOLOv8 (Deep Learning)        |
| Rastreamento          | ByteTrack                     |
| Visao Computacional   | OpenCV                        |
| Classificacao Cores   | HSV + scikit-learn            |
| Interface Grafica     | ttkbootstrap (Python)         |
| Alertas               | Telegram Bot API              |
| Relatorios            | fpdf2                         |


FUNCIONALIDADES PRINCIPAIS
--------------------------
1. Deteccao de veiculos em tempo real com YOLOv8
2. Rastreamento com ID unico por veiculo (ByteTrack)
3. Contagem bidirecional (entrada/saida)
4. Classificacao automatica de cor do veiculo
5. Analise de fluxo de trafego (veiculos/minuto)
6. Classificacao do tipo de transito
7. Sistema de alerta por cor suspeita
8. Notificacao via Telegram para GCM
9. Zoom digital para videos com veiculos distantes
10. Exportacao de relatorio PDF


DESAFIOS TECNICOS SUPERADOS
---------------------------
1. Deteccao precisa em diferentes condicoes de luz
2. Rastreamento sem perda de identidade entre frames
3. Classificacao de cores similares (prata/cinza/branco)
4. Interface responsiva durante processamento pesado
5. Alertas em tempo real sem travar o sistema
6. Adaptacao para videos verticais com zoom


RESULTADOS ALCANCADOS
---------------------
- Acuracia de deteccao: ~95%
- Processamento: 30 FPS em tempo real
- Latencia de alerta: < 2 segundos
- 10+ cores de veiculos classificadas
- 5 niveis de classificacao de transito


DIFERENCIAIS
------------
- Processamento 100% local (sem cloud)
- Tecnologias open-source (sem custos de licenca)
- Interface profissional para operadores
- Integracao com Telegram para comunicacao rapida
- Relatorios automatizados para gestao


APLICACOES
----------
- Monitoramento de trafego urbano
- Apoio a seguranca publica (GCM)
- Planejamento de mobilidade urbana
- Fiscalizacao de transito
- Coleta de dados para smart cities
